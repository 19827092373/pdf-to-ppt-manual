# PDF习题拆分PPT工具 - 用户使用指南

## 🌐 方式一：在线使用（推荐，最简单）

如果你已经将项目部署到云平台（如Render），可以直接访问网址使用，无需安装任何软件。

**访问地址**：部署后会提供（例如：`https://pdf-to-ppt.onrender.com`）

---

## 💻 方式二：本地运行（需要安装Python）

### 第一步：获取代码

**方法1：下载ZIP文件（最简单）**
1. 访问：https://github.com/19827092373/pdf-to-ppt-manual
2. 点击绿色的 "Code" 按钮
3. 选择 "Download ZIP"
4. 解压到任意文件夹

**方法2：使用Git克隆（推荐开发者）**
```bash
git clone https://github.com/19827092373/pdf-to-ppt-manual.git
cd pdf-to-ppt-manual
```

### 第二步：安装Python

1. **下载Python**
   - 访问：https://www.python.org/downloads/
   - 下载Python 3.7或更高版本
   - 安装时**务必勾选** "Add Python to PATH"

2. **验证安装**
   打开命令行（Windows按Win+R，输入cmd），运行：
   ```bash
   python --version
   ```
   应该显示Python版本号（如：Python 3.10.0）

### 第三步：安装依赖

在项目文件夹中打开命令行，运行：

```bash
pip install -r requirements.txt
```

**如果下载慢**（国内用户），可以使用国内镜像：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 第四步：启动应用

**Windows系统**：
- 双击 `run.bat` 文件
- 或在命令行运行：`python app.py`

**Linux/macOS系统**：
```bash
python app.py
```

### 第五步：使用应用

1. 打开浏览器，访问：`http://localhost:5001`
2. 上传PDF文件
3. 按照界面提示操作（详见下方使用步骤）

---

## 📖 详细使用步骤

### 1. 上传PDF文件
- 点击上传区域或拖拽PDF文件
- 等待处理完成（PDF会转换为图片）

### 2. 选择页面
- 在左侧页面列表中点击要处理的页面
- 页面大图会显示在中间区域

### 3. 选择题目
- **绘制矩形框**：在图片上点击并拖拽，绘制矩形框选择题目
- **调整位置**：点击矩形框内部并拖拽，可以移动位置
- **调整大小**：点击矩形框边缘或角落，可以调整大小
- **选中矩形框**：双击矩形框可以选中它（选中的会变成蓝色）

### 4. 管理选择
- 右侧面板显示所有已选择的题目
- 可以删除、重新选择题目
- 可以合并跨页题目（如果题目跨越多页）

### 5. 生成PPT
- 处理完所有页面后，点击"生成PPT"按钮
- 等待处理完成
- PPT文件会自动下载

## ❓ 常见问题

### Q: 提示"找不到python命令"？
A: 
1. 确认已安装Python
2. 确认安装时勾选了"Add Python to PATH"
3. 重启命令行窗口
4. 或使用 `py` 命令代替 `python`（Windows）

### Q: 提示"pip不是内部或外部命令"？
A:
1. 确认Python已正确安装
2. 尝试使用 `python -m pip` 代替 `pip`
3. 或重新安装Python并勾选"Add Python to PATH"

### Q: 安装依赖时提示"网络错误"？
A:
使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 端口5001被占用？
A:
1. 使用其他端口：`python app.py 5002`
2. 或运行 `run_port.bat` 自定义端口

### Q: 上传PDF后看不到图片？
A:
1. 检查PDF文件是否损坏
2. 查看浏览器控制台是否有错误（按F12）
3. 确认已安装所有依赖

### Q: 生成的PPT图片模糊？
A:
1. 确保PDF文件本身清晰
2. 选择题目时尽量包含完整内容
3. 如果PDF是扫描件，确保扫描分辨率足够高
