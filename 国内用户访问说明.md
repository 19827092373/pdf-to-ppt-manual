## 🚀 快速开始（国内用户）

### 推荐流程：

1. **购买服务器**
   - 阿里云轻量服务器：24元/月
   - 或学生机：9.5元/月

2. **部署应用**
   ```bash
   ssh root@your-server-ip
   cd /opt
   git clone your-repo-url pdf-to-ppt-manual
   cd pdf-to-ppt-manual
   chmod +x deploy.sh
   ./deploy.sh
   ```

3. **访问应用**
   ```
   http://your-server-ip
   ```

4. **分享给用户**
   - 将IP地址分享给国内用户
   - 所有用户都能正常访问
