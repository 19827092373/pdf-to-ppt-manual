# PPT配置说明

## 当前配置

### 1. PPT尺寸：16:9比例 ✅
- **宽度**：13.333英寸
- **高度**：7.5英寸
- **比例**：13.333 / 7.5 = 1.7778 (16:9)

### 2. 图片位置：左上角，保留边距 ✅
- **左边距**：0.5英寸
- **上边距**：0.5英寸
- **效果**：图片不会完全贴边，有适当的留白，保证排版美观

### 3. 图片大小：面积占PPT一半 ✅
- **目标面积**：PPT总面积的50%
- **计算方式**：
  - PPT总面积 = 13.333 × 7.5 = 100平方英寸
  - 目标图片面积 = 100 × 0.5 = 50平方英寸
  - 图片会按比例缩放，使面积达到50平方英寸
- **保持宽高比**：图片原始宽高比会被保持
- **边界限制**：如果缩放后超出可用区域，会进一步缩小以适应

## 示例计算

假设有一张图片：
- 原始尺寸：2000px × 1500px
- DPI：200
- 原始尺寸（英寸）：10英寸 × 7.5英寸 = 75平方英寸

计算过程：
1. PPT总面积 = 13.333 × 7.5 = 100平方英寸
2. 目标面积 = 100 × 0.5 = 50平方英寸
3. 缩放比例 = √(50 / 75) = √0.667 = 0.816
4. 最终尺寸 = 10 × 0.816 × 7.5 × 0.816 = 8.16英寸 × 6.12英寸
5. 最终面积 = 8.16 × 6.12 ≈ 50平方英寸 ✅

## 视觉效果

- PPT页面：16:9宽屏比例，适合现代投影设备
- 图片位置：左上角，距离左边和上边各0.5英寸
- 图片大小：适中，占页面一半面积，不会太大也不会太小
- 排版：美观，有适当的留白空间

## 调整参数

如果需要调整，可以修改 `utils/ppt_generator.py` 中的参数：

```python
# 边距调整
self.margin_left = Inches(0.5)    # 左边距
self.margin_top = Inches(0.5)      # 上边距

# 图片面积比例调整
self.target_area_ratio = 0.5      # 0.5表示50%，可以改为0.4（40%）或0.6（60%）
```
