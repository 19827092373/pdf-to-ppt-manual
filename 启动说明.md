# 启动说明

## 快速启动（Windows）

### 方法1：双击启动脚本（最简单）
1. 进入项目目录：`pdf-to-ppt-manual`
2. 双击 `run.bat` 文件
3. 等待服务器启动（会显示"正在启动服务器..."）
4. 打开浏览器访问：`http://localhost:5001`

### 方法2：自定义端口启动
如果端口5001被占用：
1. 双击 `run_port.bat` 文件
2. 输入你想要的端口号（如：5002、5003等）
3. 按回车启动
4. 访问：`http://localhost:你输入的端口号`

### 方法3：命令行启动
1. 打开命令提示符（CMD）或PowerShell
2. 进入项目目录：
   ```bash
   cd pdf-to-ppt-manual
   ```
3. 运行启动命令：
   ```bash
   python app.py 5001
   ```
4. 访问：`http://localhost:5001`

## 启动前检查

### 1. 确保已安装Python依赖
```bash
pip install -r requirements.txt
```

### 2. 确保已安装Poppler（PDF处理工具）
- Windows: 下载并添加到PATH
- 下载地址：https://github.com/oschwartz10612/poppler-windows/releases/

## 启动成功标志

启动成功后，你会看到类似以下信息：
```
正在启动服务器...
访问地址: http://localhost:5001
按 Ctrl+C 停止服务器
 * Running on http://0.0.0.0:5001
 * Debug mode: on
```

## 停止服务器

在命令行窗口中按 `Ctrl + C` 停止服务器

## 常见问题

### Q: 端口被占用怎么办？
A: 使用 `run_port.bat` 选择其他端口，或运行 `python app.py 5002`（替换为其他端口号）

### Q: 提示找不到python命令？
A: 确保Python已安装并添加到系统PATH环境变量

### Q: 提示缺少模块？
A: 运行 `pip install -r requirements.txt` 安装依赖

### Q: PDF无法处理？
A: 检查是否已安装Poppler工具并添加到PATH
